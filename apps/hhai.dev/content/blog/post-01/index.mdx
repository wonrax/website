This post shows you markdown features that I've integrated into this site.

## Code

I use [rehype-pretty-code](https://rehype-pretty-code.netlify.app/) for syntax
highlighting.

### Inline code

-	Inline code: `` `_variable` `` turns into `_variable`
-	Even syntax highlighted: `var x = 2{:md}{:go}` turns into `var x = 2{:go}`

### Code block

-	Show file name with: `title="example.go"`
-	Highlight specific lines with: `{3-7, 9, 15}` (highlights lines 3 to 7, 9, and 15)
-	Highlight words with: `/Zipcode/` (highlights all occurrences of `Zipcode`)
-	Show line numbers with: `showLineNumbers`

`````md
```go showLineNumbers title="example.go" {1, 3} /Zipcode string/ ... ```
`````

turns into:

```go showLineNumbers title="example.go" {1, 3} /Zipcode string/
package main

type Address struct {
	Street  string `json:"street"`
	Suite   string `json:"suite"`
	Zipcode string `json:"zipcode"`
}
```

## Image optimization

`![sample diagram](diagram.png)` creates image with multiple sizes for different
screen resolutions. If the image path starts with `./`, it will be relative with
the `mdx` file. Otherwise, it will look for images inside the
`content/blog/images` directory.

```md /feature-sm/ /lg/
![sample scenery](scenery.jpg)
![sample scenery|feature-sm](scenery.jpg)
![sample scenery|feature-lg](scenery.jpg)
```
